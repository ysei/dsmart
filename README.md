# manager

## 使い方

managerフォルダで，以下を実行します．

    $ rackup

初回の実行前に，以下が必要かも．

    $ bundle 

## ブラウザでのアクセス

データベースの登録と，状態確認ができます．

<http://localhost:9292/list> でデータベースの状態を確認できます．

<http://localhost:9292/add_server?table_name=aaa&url=server1> でサーバを追加します．
table_nameにテーブル名を，urlにサーバ名を指定します．
